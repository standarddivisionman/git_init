# git_init
1. open your terminal
2. paste this: "git clone https://github.com/standarddivisionman/git_init.git"
3. change directory to git_init with this ("cd git_init")
4. install snakemake, if you have it not: pip install snakemake
5. paste "snakemake"(without quotes) to run snakefile(you may face massive problems here, but it's not my problem)
6. enjoy
